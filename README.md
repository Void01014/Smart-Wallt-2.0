Smart Wallet

Smart Wallet is a web application that helps users manage their money by tracking transactions, income, and expenses.

Features

User registration and login using email and password

OTP authentication for user verification

Send money to other users using:

Email

User ID

Add income and expenses with:

Category

Description

Amount

Date

Automate income or expense entries:

Daily

Weekly

Monthly

Dashboard showing:

Current balance

Total income

Total expenses

Profile page with:

Account ID

Username

Email

Number of connected accounts/cards

Technologies Used

HTML, CSS, JavaScript

PHP

MySQL
